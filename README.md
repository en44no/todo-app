## 📝 Features

>- Light/dark mode toggle
>- Create tasks
>- Remove tasks
>- Edit tasks
>- List tasks
>- Mark as completed and pending toggle
>- Filter tasks
>- Notifications
>- Checks for task creation and editing


## 💻 Run Locally

>Clone the project
>
>```bash
>  git clone https://github.com/en44no/todo-app.git
>```
>
>Go to the project directory
>
>```bash
>  cd my-project
>```
>
>Install dependencies
>
>```bash
>  npm install
>```
>
>Start the server
>
>```bash
>  npm start
>```
